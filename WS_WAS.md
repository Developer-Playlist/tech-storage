<h3>WS와 WAS</h3>
WS : Web Server / IIS, nginx 등
WAS : Web Application Server / Django, Flask 등 

<h3>WS의 기능 (전반적인)</h3>
- 인증
- 정적 콘텐츠 관리
- HTTPS 지원
- 콘텐츠 압축
- 가상 호스팅
- 대용량 파일 지원
- 대역폭 스로틀링
